## Project Name
Uni project - Students management system

## Project Description
Java GUI program that allows to manage students

## Project Features
Enroll new students<br>
View all students<br>
Pay students fees<br>
View student profile<br>
Change student course<br>
Remove student<br>


## Prerequisites
Java 1.6
